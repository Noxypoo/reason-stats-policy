# Privacy Policy

**Reason Stats Bot**
Last updated: April 13, 2026

---

## 1. What Data We Collect

When you use Reason Stats Bot, we collect and store the following data:

- **Discord User ID** — stored when you register for vote reminders via `/vote add`
- **In-game username** — the username you provide with `/vote add`
- **Discord Server (Guild) ID** — stored when a server admin configures the bot via `/setup vote create`
- **Vote reminder channel ID** — the channel configured per server

We do **not** collect or store:
- Discord usernames or display names
- Message content
- Any data beyond what is listed above

---

## 2. How We Use Your Data

The data listed above is used solely to:
- Send you vote reminder notifications in the configured server channel
- React to reminder messages when a vote is detected

We do not sell, share, or use your data for any other purpose.

---

## 3. Data Storage

All data is stored in a private MongoDB database. Access is restricted to the bot operator only. Data is not shared with any third parties.

---

## 4. Data Retention

Your data is retained for as long as you remain registered with the bot. You can remove your data at any time using `/vote remove`, which permanently deletes your Discord User ID and in-game username from our database.

Server configuration data is retained as long as the bot remains in the server. Server admins can remove it at any time using `/setup vote remove`.

---

## 5. Your Rights

You have the right to:
- **Access** — request a copy of the data we hold about you
- **Delete** — remove your data at any time via `/vote remove`

For requests or questions, contact the bot operator via the ReasonPS Discord server.

---

## 6. Changes to This Policy

We may update this policy from time to time. Continued use of the bot after changes constitutes acceptance of the updated policy.
